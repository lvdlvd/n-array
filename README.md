# [N]Array - write to bare ARMs

[N]array is a set of tools to create bare metal firmware in C for ARM Cortex M based microcontrollers of the STM32 G4 families.

[N]Array shuns premature abstraction.  The STM32 Reference and Programming Manuals, the Datasheets and the Errata are ground truth documents.  There is no operating system, the traditional tasks of the operating system, managing CPU, memory, i/o and storage are implemented directly on the device hardware primitives.   [N]Array provides readable header files to use the builtin functions of the device directly, generated by parsing the reference documentation.  Options are configured by including the appropriate header 

Building a [N]Array project requires only the [arm-none-eabi GCC toolchain](https://developer.arm.com/downloads/-/arm-gnu-toolchain-downloads).

## Features

- [ ] don't postpone till runtime what can be safely determined at compile time
- [ ] no conditional compilation, no transitive includes, no dead code
- [ ] define pin mappings
- [ ] run from RAM or from ROM
- [ ] static/dynamic vector table: fault handler, boot handler ...
- [ ] configure clock
- [ ] configure irq priorities
- [ ] configure dma channels
- [ ] optional drivers for serial, fdcan, usb, spi, i2c, flash, timers
- [ ] buffers, tprintf, binary encoding, can messages, diagnostic timers
- [ ] support for unpriviledged processes?

## Embedded System Design Workflow

- define system requirements
    - functional: what signals come out, what signals go in, what should the system do?
    - non-functional: speed, accuracy, size, weight, power, reliability
- define ICD: physical/software interfaces, connectors, data dictionaries 
- define software requirements
    - modes, states, ..
- define hardware requirements
    - choose external i/o peripherals
    - choose external memory peripherals
    - choose STM32 family and model
    - define pinout configuration as the interface between sw/hw
- create boilerplate application
- optional create test harness application (based on same STM32 MCU)

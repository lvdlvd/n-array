# Concrete - write to bare ARMs

Concrete is a set of tools to create bare metal firmware in C for ARM Cortex M based microcontrollers of the STM32 L4, G0, G4, and H7 families.

Concrete shuns premature abstraction.  The STM32 Reference and Programming Manuals, the Datasheets and the Errata are ground truth documents.  There is no operating system, the traditional tasks of the operating system, managing CPU, memory, i/o and storage are implemented directly on the device hardware primitives.   Concrete provides header files and minimal libraries to use the builtin functions of the device directly, generated by parsing the reference documentation.
